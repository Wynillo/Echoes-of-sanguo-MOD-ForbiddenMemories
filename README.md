# Echoes-of-sanguo-MOD-ForbiddenMemories

A **Yu-Gi-Oh! Forbidden Memories** MOD for the [Echoes of Sanguo](https://github.com/Wynillo/Echoes-of-Sanguo-TCG) engine.

## Contents

- **722 cards** — All cards from Yu-Gi-Oh! Forbidden Memories with accurate stats (ATK/DEF/Level/Type/Attribute)
- **142 fusion formulas** — Type-based fusion system matching the original FM fusion mechanics
- **39 opponents** — Full roster from Simon Muran to Nitemare
- **5-chapter campaign** — Ancient Egypt storyline with 31 duels
- **6 booster packs** — Progressive shop system using Starchips
- **6 starter decks** — Dragon, Spellcaster, Warrior, Beast, Fiend, and Machine themed
- **20 monster types** — Dragon, Spellcaster, Warrior, Beast-Warrior, Beast, Winged Beast, Fiend, Zombie, Aqua, Pyro, Rock, Plant, Insect, Fish, Sea Serpent, Reptile, Machine, Thunder, Dinosaur, Fairy
- **English & German** locales

## Build

```bash
npm install
npm run build:tcg    # -> dist/base.tcg
npm run validate     # validate without building
```
